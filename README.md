## Rick and Morty
Rick & Morty character information
### Built with:
- [React]
- [Tailwindcss]
### Installation and setup
1. Clone this repo
```bash
gh repo clone emrocode/rick-and-morty
```
2. Go to folder and install dependencies
```bash
npm install
```
3. Run
```bash
npm run dev
```
